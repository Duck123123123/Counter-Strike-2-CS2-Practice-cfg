# Counter-Strike-2-CS2-Practice-cfg
This cfg setup everything for you, however this cfg have keybind included like noclip, spawn bot ect...
you can edit these key bind to whatever you like.
line 48-56 is where you edit your key bind.

How to use?
find your cs2 where is installed, location is SteamLibrary\steamapps\common\Counter-Strike Global Offensive\game\csgo\cfg
this is where you put 1.cfg
in game make sure you have developer console on then choose any practice map you want and get in game
onec in game open console type exec 1
